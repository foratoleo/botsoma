# Helpbot Knowledge Base

Created: 2026-04-10
Last Updated: 2026-04-10

## Overview

The `helpbot/` directory contains 11 Markdown files that serve as a knowledge base for a Telegram bot. These documents explain the WORKFORCE platform to end-users in Portuguese (pt-BR).

## Content Summary

### 01 - Visao Geral
Platform introduction, purpose, key features, user roles.

### 02 - Navegacao e Areas
Navigation system, 4 workflow areas, area detection, sidebar behavior.

### 03 - Gestao de Projetos
Project CRUD, creation wizard, project selector, project details.

### 04 - Gestao de Equipes, Tarefas e Sprints
Team management, member roles, task lifecycle, sprint planning, kanban board.

### 05 - Transcricoes e Reunioes
Meeting creation, recording, transcription, Recall.ai integration.

### 06 - Geracao de Documentos com IA
AI document generation pipeline, supported types, Edge Functions, approval workflow.

### 07 - Chat, RAG e Guru
RAG system, chat functionality, knowledge base, source tracking.

### 08 - Upload de Documentos
File upload, supported formats, processing pipeline, PDF extraction.

### 09 - Arquitetura Tecnica
Tech stack, data flow, component architecture, security.

### 10 - Integracoes
Jira sync, GitHub PRs, Microsoft Calendar, Recall.ai.

### 11 - Perguntas Frequentes
Common questions, troubleshooting, tips.

## Usage

These files are designed to be loaded as context for a Telegram bot that answers user questions about the WORKFORCE platform. The bot can:
- Search across all files for relevant information
- Provide step-by-step instructions for common tasks
- Explain features and workflows
- Troubleshoot common issues

## Language

All helpbot files are written in Portuguese (pt-BR) to match the primary user base.
